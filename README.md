# Add Cypress

- setup and installation
  ```shell
  $ yarn add cypress -D
  ```
- It will install the Cypress Desktop App & CLI
- Now update your package.json file
  ```json:package.json
  {
    "scripts": {
      "e2e": "cypress open"
    },
  }
  ```
- When you run it for the first time it will provide a bunch of examples
